## 插件说明
基于 siliconflow 提供的生图功能。地址: [[https://cloud.siliconflow.cn/i/0dhpY8io](https://cloud.siliconflow.cn/i/0dhpY8io)]，注册有2000万 Tokens

## 插件配置
维护目录下的 `config.json` 。

```bash
{
  "siliconflow_api_key": "sk-xxx"
}

```
